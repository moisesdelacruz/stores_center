# S-Center (Stores Center)
It's a application built on Django.

## Requirements OS
- Postgresql
- NPM.js
- Virtualenv

## Install
```sh
  git clone https://github.com/moisesdelacruz/stores_center.git

  cd stores_center/

  pip install requirements.pip

  npm install

  npm start

  python manage.py migrate

  python manage.py runserver
```
